
# Bitcoin Analytics

Analysis of Bitcoin network usage from 2009 to 2017 using 492,000 blocks from a Bitcoin Core node.

## Project Description

This project analyzes how Bitcoin usage grew during its first eight years by examining block data. Key metrics include transaction volume, block size, empty block rate, block intervals, and the impact of SegWit and halving events.
